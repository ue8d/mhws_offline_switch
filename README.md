# MHWS Offline Switch

モンスターハンターワイルズ（と推測されるゲーム）のオンライン接続とオフラインモードを簡易的に切り替えるためのツールです。

## 使い方

1.  `WildsNetSwitch.config.json` をお使いの環境に合わせて編集します。
2.  `wilds_net_switch.py` を実行して、表示される指示に従ってください。
